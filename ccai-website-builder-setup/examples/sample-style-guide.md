## Color palette

### Primary
- **HEX:** `#0a0a0a` (near-black)
- **Tailwind class:** `bg-primary` / `text-primary`
- **Usage:** main brand color, CTAs, key headings, body text

### Secondary
- **HEX:** `#fafaf8` (warm off-white)
- **Tailwind class:** `bg-secondary` / `text-secondary`
- **Usage:** main background, soft sections

### Accent
- **HEX:** `#dc2626` (clear red)
- **Tailwind class:** `bg-accent` / `text-accent`
- **Usage:** sparingly, for the "wrong" or "warning" visuals (strikethrough on Free plan, callouts for failure stats)

### Neutral palette
- Background: `bg-neutral-50`
- Text body: `text-neutral-900`
- Text muted: `text-neutral-600`
- Borders: `border-neutral-200`

---

## Typography

### Heading font
- **Family:** DM Serif Display (for editorial weight on H1/H2)
- **Loaded via:** Google Fonts via next/font
- **Weights:** 400 only
- **Tailwind class:** `font-serif`

### Body font
- **Family:** DM Sans
- **Loaded via:** next/font
- **Weights:** 400, 500, 600
- **Tailwind class:** `font-sans` (default)

### Code/numeric
- **Family:** DM Mono (for stats, numbers, code samples)
- **Tailwind class:** `font-mono`

### Scale
- H1: `text-5xl md:text-6xl font-serif font-normal` (DM Serif at 48-60px)
- H2: `text-3xl md:text-4xl font-serif font-normal`
- H3: `text-2xl font-sans font-semibold`
- Body: `text-base font-sans`
- Small: `text-sm font-sans text-neutral-600`

---

## Spacing

Tailwind default 4px scale.
- Section vertical padding: `py-20 md:py-28`
- Container max-width: `max-w-5xl`
- Container horizontal padding: `px-6 md:px-8`

---

## Components

### Buttons
- Primary CTA: `<Button variant="default" size="lg">`, solid black, white text
- Secondary: `<Button variant="outline" size="lg">`, black border on white background
- Ghost: `<Button variant="ghost">` for nav links

### Cards
- Use shadcn `<Card>` for testimonial blocks, feature cards
- Border: `border-neutral-200`
- Radius: `rounded-md`
- Shadow: NONE, flat by default, matches the editorial brand

### Forms
- shadcn `<Input>` with always-visible labels
- Submit button: full-width on mobile, auto-width on desktop
- Validation message below input, in `text-accent text-sm`

## Animation

### Default page entry
- `initial={{ opacity: 0, y: 16 }}`
- `animate={{ opacity: 1, y: 0 }}`
- `transition={{ duration: 0.5, ease: [0.22, 1, 0.36, 1] }}`

### Scroll reveals
Used sparingly. Only on testimonials and the proof-anchor stats. Once-only, viewport-triggered.

### Banned animations
- No bounces, springs, or zoom-ins
- No animated text, fades on the container only
- Hero entry is the only animation on initial page load; everything else is static

## Voice (from BRAND_VOICE.md)

### Signature vocabulary
- "Here's the honest answer:"
- "Here's the thing"
- "real" (used adjectivally)
- "actually" (as a contrarian softener)

### Taboos (do NOT use anywhere on the site)
- ❌ "transform" / "transformative"
- ❌ "revolutionary"
- ❌ "game-changer"
- ❌ "unlock the power of"
- ❌ "leverage" (as a verb)
- ❌ Emoji in long-form copy or page sections
- ❌ "Click here" CTAs
- ❌ "Don't miss out" / "Limited time" (unless literally true)

### Sentence cadence
- Average sentence length: 14 words
- Fragments allowed (1 per ~8 sentences, for emphasis)
- Em-dash usage: ~1.4 per paragraph (frequent, calibrate)

### CTA phrasing
- ✅ "Start the free course →"
- ✅ "Book a free diagnostic call"
- ✅ "Read the full breakdown"
- ❌ "Click here to learn more"
- ❌ "Sign up today!"

## Accessibility defaults

- Focus states use a 2px accent-colored ring
- Body text contrast ≥ 7:1 (exceeds WCAG AAA)
- All images have alt text; decorative use `alt=""`
- `prefers-reduced-motion` respected, animations disabled if user prefers

---

## When building a new page or section

1. Read this file
2. Read `BRAND_VOICE.md` for copy decisions
3. Use shadcn components, extend, don't replace
4. Mobile-first at 375px
5. Light mode only (dark mode not enabled for this site, CCAI brand is light-bg)
